Все наработки лежат в папке Source. Небольшие настройки в GameConfig. 
На сцене из наработок:
Picture - само полотно. на котором рисовать. Внутри него есть канвас с PokeInteractable, чтобы палец не проходил насквозь полотна (не придумал ничего лучше).
Systems - внутри две системы: Стартер - он раздаёт всем данные и настройки, а так же поддерживает Update. И Picture Load Save - отдельный скрипт для сохранения и загрузки рисунка.
UI Canvas Panel - внутри лежит UI с палитрой и кнопками. Работает от нажатия пальца (не Ray система).
